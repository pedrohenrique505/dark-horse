Add custom game sounds in this folder using these filenames:

- move.mp3
- capture.mp3
- check.mp3
- castle.mp3
- game-end.mp3

The current client will try to play these files first.
If a file is missing or blocked, it falls back to the built-in generated tone.

Recommended formats:
- mp3 for simplicity and browser support
- wav if you prefer uncompressed assets

Keep clips short and dry:
- move: short click
- capture: sharper impact
- check: alert tone
- castle: distinct special move cue
- game-end: longer finish cue
